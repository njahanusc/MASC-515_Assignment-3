# MASC 515 Assignment 3 - microGPT

## Overview
This assignment uses microGPT, a simple Python implementation of a GPT-like model created by Andrej Karpathy.

## Algorithms Implemented

### 1. GELU (Gaussian Error Linear Unit)
GELU is an activation function used in neural networks. It helps the model learn complex patterns by smoothly activating neurons.

### 2. LoRA (Low-Rank Adaptation)
LoRA is a technique to make large models more efficient by updating only small parts of the network instead of the entire model.

### 3. RoPE (Rotary Positional Embedding)
RoPE helps the model understand the order of words in a sequence by encoding position information.

### 4. Mixture of Experts (MoE)
MoE uses multiple smaller models ("experts") and selects the best one for each input, improving efficiency.

## Notes
The original microGPT code was used without modification, as required.
